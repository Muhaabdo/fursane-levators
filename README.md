# موقع مصاعد فرسان — دليل التشغيل

```
/lp/
├── index.html            الصفحة الرئيسية
├── maintenance.html      الصيانة والطوارئ
├── villa-elevators.html  مصاعد الفلل والمنازل
├── projects.html         المشاريع التجارية والإدارية
├── privacy.html          سياسة الخصوصية (noindex)
├── styles.css            الأنماط المشتركة
├── main.js               السكربت المشترك
└── assets/
    ├── fursan-logo.svg       اللوجو بلونين
    └── fursan-logo-mono.svg  لون واحد — للفافيكون والمقاسات الصغيرة
```

الموقع HTML و CSS و JavaScript خام — بدون build ولا مكتبات خارجية.
ارفعه على أي استضافة وهو شغّال.

## ثلاث خطوات قبل النشر

### ١. معرّف GTM
ابحث عن `GTM-XXXXXXX` في الصفحات الأربع واستبدله بمعرّف الحاوية.

```bash
grep -rl "GTM-XXXXXXX" *.html
```

### ٢. رابط استقبال النموذج
في أول `main.js`:

```js
const CONFIG = {
  phone: '966536600095',
  formEndpoint: '',        // ← ضع هنا رابط Formspree أو Web3Forms
  endpointType: 'json',    // 'json' أو 'formdata' حسب المزود
};
```

**ما دام `formEndpoint` فارغًا، النموذج يشتغل عبر واتساب** — يفتح المحادثة
بالبيانات مكتوبة جاهزة. يعني الصفحة تجيب عملاء من أول يوم حتى قبل ربط الإيميل.
وإن فشل الإرسال لأي سبب بعد الربط، يتحوّل تلقائيًا لواتساب بدل ضياع العميل.

### ٣. الصور
كل مكان صورة معلّم بـ `<div class="ph" data-label="...">`. استبدله بـ:

```html
<img src="assets/projects/villa-narjis.jpg" alt="مصعد فيلا في حي النرجس" loading="lazy">
```

### ٤. بروفايل الشركة PDF
صفحة المشاريع فيها ثلاثة أزرار تحميل تشير إلى:
`assets/fursan-company-profile.pdf`

ضع الملف هناك. ويُفضّل قبل الرفع إزالة الجزء الخاص بأنظمة الطاقة الشمسية
ومحطات ضخ المياه من البروفايل — يشتّت في سياق المصاعد.

---

ابحث عن `🔸` في الملفات — كل البنود المعلّقة على تأكيد العميل معلّمة بها.

## أحداث التتبع الجاهزة في dataLayer

اربطها في GTM كـ Custom Event triggers:

| الحدث | متى يُطلق |
|---|---|
| `click_call` | أي ضغط على رقم الهاتف |
| `click_whatsapp` | أي ضغط على واتساب |
| `form_submit` | إرسال نموذج الرئيسية أو الصيانة |
| `form_submit_inspection` | إرسال نموذج حجز المعاينة |
| `form_submit_project` | إرسال نموذج عرض سعر المشروع |
| `download_profile` | تحميل بروفايل الشركة PDF |
| `click_service_maintenance` | ضغط كارت الصيانة في الرئيسية |
| `click_service_villa` | ضغط كارت الفلل في الرئيسية |
| `click_service_projects` | ضغط رابط المشاريع في الرئيسية |
| `cookie_consent_granted` | الموافقة على الكوكيز |

كل حدث يحمل `page` و `location` لمعرفة مكان الضغط بالضبط.

> الحدثان `click_service_*` هما أرخص بحث سوق: يقولان أي نية تصل للرئيسية أكثر،
> وبناءً عليهما توزّع الميزانية بين حملة الصيانة وحملة الفلل.

## ملاحظات تقنية

- `dir="rtl"` و `lang="ar"` مضبوطان، وكل التباعد بخصائص منطقية (`margin-inline`)
- بدون `letter-spacing` على العربي، و`font-synthesis: none` لمنع الـbold الوهمي
- ارتفاع السطر ١.٩ للنصوص و١.٤٥ للعناوين
- `prefers-reduced-motion` محترم — الحركة تتوقف بالكامل
- تركيز لوحة المفاتيح ظاهر (`:focus-visible` ذهبي)
- الأرقام داخل `.num` بـ`direction: ltr` عشان ما تنعكس في السياق العربي
- على الموبايل: شريط سفلي ثابت باتصال وواتساب بدل الأزرار العائمة
- Schema مضاف: `Organization` + `LocalBusiness` بفرعين + `FAQPage`

## شريط الكوكيز

يستخدم كوكي `fursan_consent` لمدة ١٨٠ يومًا. لو حبيت تؤجّل تحميل سكربتات
التتبع لحد الموافقة، اربط تشغيلها في GTM بحدث `cookie_consent_granted`.
